# ATLANTIS BEVCO — WEB STYLE & BRAND SYSTEM
Version: 1.0  
Status: Locked  
Audience: AI Code Assistants (Claude, Cursor)

This document defines the complete visual, typographic, and tonal system for the Atlantis Bevco website.
Deviation is not permitted unless explicitly stated.

---

## 1. BRAND INTENT

Atlantis Bevco is a premium, authoritative beverage distributor.
The brand must communicate:
- Trust
- Longevity
- Operational competence
- Taste authority
- Calm confidence

This is NOT a lifestyle brand.
This is NOT playful.
This is NOT experimental.

---

## 2. COLOR SYSTEM

### Primary Colors (Greek Flag Inspired)

--color-atlantis-blue: #0B4F9F;
--color-white: #FFFFFF;

Usage:
- These two colors must represent 70–80% of the UI.
- Backgrounds default to white.
- Headers, footers, and navigation default to Atlantis Blue.

---

### Accent Color (Single Accent Only)

--color-aegean-gold: #E0B44C;

Usage rules:
- Accent color must NOT exceed 10–15% of any page.
- Allowed uses:
  - Primary CTA buttons
  - Section dividers
  - Icon highlights
  - Data emphasis
- Never use for body text.
- Never use for backgrounds larger than a button or rule.

---

### Neutral Support Colors (UI Only)

--color-slate-navy: #1F2A37;
--color-cool-gray: #6B7280;
--color-light-gray: #E5E7EB;

---

## 3. TYPOGRAPHY

### Font Stack

#### Primary Serif (Headlines)
Preferred:
- "Cinzel", serif

Fallback:
- "Playfair Display", serif

Usage:
- Page titles
- Section headers
- Hero headlines
- Feature callouts

---

#### Secondary Sans-Serif (Body/UI)
Primary:
- "Inter", system-ui, sans-serif

Usage:
- Body copy
- Tables
- Forms
- Navigation
- Footer text

---

### Type Scale

H1: Serif, 48–56px, Bold  
H2: Serif, 36–40px, Semibold  
H3: Sans, 24–28px, Semibold  
Body: Sans, 16–18px, Regular  
Small: Sans, 14px  

Rules:
- Never mix more than one serif and one sans.
- Never stylize body copy.
- No all-caps body text.

---

## 4. LAYOUT SYSTEM

### Grid
- Max width: 1200px
- Side padding: 24px desktop / 16px mobile
- Section spacing: generous vertical rhythm

### Layout Principles
- Symmetry over chaos
- White space is mandatory
- Content must breathe
- No cluttered blocks

---

## 5. BUTTONS & UI ELEMENTS

### Primary Button
Background: Aegean Gold  
Text: Atlantis Blue  
Border Radius: 4–6px  
Font: Sans, Semibold  

### Secondary Button
Background: Transparent  
Border: 1px solid Atlantis Blue  
Text: Atlantis Blue  

Hover:
- Subtle darkening only
- No animations beyond 150ms

---

## 6. ICONOGRAPHY

- Simple, geometric icons only
- No rounded cartoon icons
- Prefer outline or solid, not mixed
- Icon color:
  - Atlantis Blue
  - White
  - Aegean Gold (sparingly)

---

## 7. IMAGERY

Photography rules:
- Editorial, real-world
- Natural lighting
- Hospitality environments
- Bottles, back bars, logistics, people at work

Avoid:
- Stock "cheers" photos
- Influencer aesthetics
- Heavy filters
- Illustrations

---

## 8. COPY & TONE

Voice attributes:
- Confident
- Measured
- Informed
- Calm authority

Avoid:
- Hype language
- Slang
- Emojis
- Exclamation points

Examples:
❌ “Game-changing portfolio”
✅ “A curated portfolio built for consistent execution”

---

## 9. NAVIGATION & UX

- Navigation must be clear and restrained
- No sticky gimmicks
- No parallax
- No scroll hijacking
- Performance > animation

---

## 10. GOVERNANCE (CRITICAL)

Do NOT:
- Introduce new colors
- Introduce gradients
- Use neon or saturated tones
- Add decorative fonts
- Animate excessively
- Reinterpret the brand creatively

This system is intentional and final.
